## User Roles

## Entry Conditions

## State Model

## User Journeys

## API Contracts

List the current Next route or proxy path, HTTP method, request shape, response shape, and the
touchpoint files that currently implement the contract.

## Server Touchpoints

List the server-side files, proxy helpers, cache helpers, or document/photo services touched by this
module.

## Performance Guardrails

and invalidation or refresh triggers.

## Invariants

## Failure Modes

Describe network, validation, cache, and mutation failures that the module must preserve.

## Industry Variability

## Composition Examples
