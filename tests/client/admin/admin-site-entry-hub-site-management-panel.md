# Admin Proof: Site Management — Optional Edit Action

## Expected Behavior

- in `/admin?section=headquarters` with a site selected, the site main panel header shows `현장 정보 수정` when the default props are used
- the edit link navigates to the existing site edit route for that site

## Manual Check

- open admin headquarters drilldown and select a site
- confirm `현장 정보 수정` is visible and opens the expected edit flow
