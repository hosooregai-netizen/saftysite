# 02. MVP 범위 축소

```md
좋아. 방금 계획을 바탕으로 MVP 1차 범위만 다시 정리해줘.

MVP 1차 범위:
- 최소 사진 업로드 구조 정리
- PhotoObservationCard 타입/스키마 추가
- 위험요인 사진 1장으로 4번 현재 공정 내 현존하는 위험성 제거 초안 생성
- 공정 사진 1장으로 5번 향후 진행공정 위험요인 및 대책 초안 생성
- 기존 HWPX/PDF 생성은 건드리지 말고 payload mapping 가능성까지만 확인

이번 MVP에서 제외:
- 이전 지적사항 이행여부 자동판단 고도화
- 교육/지원사항 고도화
- 메일 발송 변경
- 관리자 대시보드 변경
- 대규모 DB 마이그레이션
- HWPX/PDF 라우트 대규모 수정

다시 구현 순서를 5~7개 작은 작업으로 나눠줘.

각 작업마다:
- 목표
- 수정 파일 후보
- 입력/출력 shape
- 테스트 방법
- 롤백 위험도
를 적어줘.

아직 코드는 수정하지 마.
```
