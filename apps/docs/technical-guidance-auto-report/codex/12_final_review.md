# 12. 최종 검토

```md
전체 변경사항을 최종 검토해줘.

반드시 읽을 문서:
- AGENTS.md
- docs/technical-guidance-auto-report/00_index.md
- docs/technical-guidance-auto-report/reference/standard_report_structure.md

검토 기준:
- AI가 사실정보를 생성하지 않는지
- 사진 관찰값이 PhotoObservationCard로 분리되어 있는지
- 보고서 문장이 표준 위험 라이브러리에서 생성되는지
- 4번 현재 위험성 제거가 표준보고서 구조에 맞는지
- 5번 향후 진행공정 위험요인 및 대책이 표준보고서 구조에 맞는지
- review queue가 필요한 항목을 잡는지
- 기존 guided photo flow가 깨지지 않았는지
- 기존 HWPX/PDF 생성이 깨지지 않았는지
- local mode가 유지되는지

가능한 모든 테스트/타입체크/빌드를 실행해줘.

마지막에 다음 형식으로 요약해줘:
1. 구현 완료 항목
2. 변경 파일
3. 실행한 명령과 결과
4. 남은 리스크
5. 다음 단계 추천
```
