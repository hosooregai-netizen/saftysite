# Step 28 Manifest: Release Blocker Patch Template Pack

Step 27 release / hold decision 이후 실제 blocker가 발생했을 때 바로 patch sprint를 시작할 수 있도록 기능별 blocker patch 템플릿과 실행 프롬프트를 제공한다.

## 포함 범위

- blocker intake template
- S0~S4 severity strategy
- focused QA matrix
- patch verification checklist
- feature별 blocker patch prompt
- blocker resolution report template
