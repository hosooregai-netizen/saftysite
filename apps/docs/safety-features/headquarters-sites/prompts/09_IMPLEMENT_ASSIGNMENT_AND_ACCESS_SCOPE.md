# 09_IMPLEMENT_ASSIGNMENT_AND_ACCESS_SCOPE

```text
너는 사업장/현장 사용자 배정과 접근 범위를 구현하는 시니어 백엔드/프론트엔드 엔지니어다.

참조 문서:
- docs/safety-features/headquarters-sites/specs/assignment_access_scope_hardening.md
- docs/safety-features/auth-workspace/specs/workspace_access.md

요구사항:
1. 관리자/일반 사용자 접근 범위를 구분하라.
2. 사업장 배정과 현장 배정 UI를 정리하라.
3. 미배정 사용자는 빈 목록 또는 안내 state를 표시하라.
4. 다른 workspace 사용자/현장/사업장 배정을 차단하라.
5. assignment 변경 후 목록을 갱신하라.

완료 기준:
- 사용자별 배정 범위가 정확히 적용된다.
- 권한 없는 사용자는 assignment 관리 UI를 사용할 수 없다.
```
