# Headquarters/Sites Visual QA After Recovery

## 기준

`headquarters-sites`는 ERP 기준정보 관리 화면이다.

## Desktop

- 좌측 ERP AppShell 유지
- page header card
- 검색/필터 toolbar
- 사업장 table/list
- 현장 table/list
- 상세 panel 또는 modal
- quick actions

## Non-regression

- fullscreen Drive/Mail shell로 전환하지 않는다.
- nested card를 과하게 사용하지 않는다.
- 사업장/현장과 사진첩/보고서 링크가 끊기면 안 된다.
