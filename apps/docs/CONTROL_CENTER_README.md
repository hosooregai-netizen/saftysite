# Control Center Added

이 문서 패키지는 기존 Markdown 구조에 `docs/control-center/`를 추가한다.

## 왜 추가했나

- 기존 Markdown은 원본 명세로 유지한다.
- HTML은 사람이 적용 순서, 프롬프트, QA, blocker를 빠르게 조작하기 위한 인터페이스다.

## 열기

```text
docs/control-center/index.html
```

- features.json
- service_improvements.json
- prompts.json
- routes.json
- qa_gates.json
- blockers.json
