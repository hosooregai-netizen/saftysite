# 06_VISUAL_POLISH: Photo Album

```text
너는 ERP 사진첩 화면의 UI/UX를 개선하는 시니어 디자이너/프론트엔드 엔지니어다.

목표:
사진첩을 현장 사진 보관/조회 서비스답게 보기 좋고 사용하기 쉽게 다듬어라.

요구사항:
1. ERP AppShell 패턴을 유지하라.
2. 사진첩 header와 upload CTA를 명확히 하라.
3. filter bar를 간결하게 정리하라.
4. photo card는 thumbnail, 현장명, 회차, 촬영일, 파일명을 표시하라.
5. 선택 toolbar와 preview panel을 제공하라.
6. empty state는 icon + 설명 + CTA 구조로 작성하라.
7. nested card와 과한 shadow를 피하라.
8. 모바일은 2-column 또는 1-column grid로 대응하라.
```
