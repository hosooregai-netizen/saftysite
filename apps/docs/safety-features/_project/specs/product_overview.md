# Product Overview

## 제품명

대한안전산업연구원 업무지원 시스템

## 제품 목적

기술지도 결과보고서 작성, 사업장/현장 기준정보 관리, 사진 보관, 자료 공유, 업무 메일, 결제/크레딧을 하나의 SaaS/ERP형 업무 환경에서 제공한다.

## 핵심 사용자

| 사용자 | 주요 니즈 |
|---|---|
| 기술지도 실무자 | 보고서 작성, 현장 사진 정리, 파일 공유 |
| 현장/사업장 관리자 | 기준정보 관리, 자료 확인, 메일 송수신 |
| 내부 관리자 | 계정/결제/크레딧/워크스페이스 관리 |

## 주요 기능

1. 보고서 작성 / AI 자동작성
2. 보고서 목록 / 출력 이력
3. 사업장/현장 기준정보
4. 사진첩
5. 웹하드
6. 메일함
7. 계정/설정
8. 결제/크레딧
9. 인증/워크스페이스

## 문서화 목표

- 기능별 명세와 구현 프롬프트를 분리한다.
- 기능을 나중에 다시 구현할 수 있도록 route, component, API, data model, prompt를 역추적한다.
- 웹하드와 메일함처럼 독립 앱에 가까운 화면은 ERP AppShell이 아니라 full-screen workspace pattern으로 문서화한다.
