# 12_LINKED_FEATURE_NAVIGATION

```text
너는 사업장/현장 기준정보에서 다른 업무 기능으로 이동하는 quick action을 정리하는 시니어 프론트엔드 엔지니어다.

참조 문서:
- docs/safety-features/headquarters-sites/specs/linked_feature_navigation_hardening.md

요구사항:
1. 현장 상세에서 새 보고서 작성 링크를 제공하라.
2. 현장/사업장 사진첩 링크를 제공하라.
3. 보고서 목록 필터 링크를 제공하라.
4. 메일함 context link를 제공하라.
5. 링크에는 headquarterId/siteId query를 정확히 포함하라.

완료 기준:
- 기준정보 화면에서 보고서 작성/사진첩/보고서 목록으로 자연스럽게 이동한다.
```
