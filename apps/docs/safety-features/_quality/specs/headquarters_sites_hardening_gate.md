# Headquarters/Sites Hardening Gate

## Release blocking

- `/headquarters` 또는 `/sites` build/route smoke 실패
- source readiness import 누락 재발
- 다른 workspace 사업장/현장 접근 허용
- 일반 사용자가 assignment 관리 가능
- 보고서 작성/사진첩 링크 query 오류
- guest/auth mode action 혼동
- ERP AppShell이 아닌 잘못된 layout 회귀

## Non-blocking

- assignment modal이 MVP 수준
- 엑셀 export 미구현
- 현장 상세 통계 일부 placeholder
