# Data Table Pattern

## 사용 기능

- report-list
- headquarters-sites
- webhard list view
- billing ledger

## Column rules

- `name/title`: 가장 넓은 column
- `status`: badge
- `date`: 고정 너비
- `actions`: 우측 고정
- 모바일에서는 column collapse

## Interaction

- row hover
- selected row
- keyboard navigation
- sort indicator
- empty state
- loading skeleton

## 금지

- table 안에 과한 카드 중첩
- 모든 데이터를 작은 badge로 만드는 것
- row action이 hover 때만 보여서 접근성이 떨어지는 것
