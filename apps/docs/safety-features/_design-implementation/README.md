# Feature Design Implementation Prompts

이 폴더는 기능별 디자인 구현 프롬프트의 색인과 공통 규칙을 관리한다.

## 왜 필요한가

기존 문서에는 기능 명세, 데이터 흐름, schema/API, QA 프롬프트가 있었지만, 실제 UI 구현자가 바로 사용할 수 있는 **디자인 구현 프롬프트**가 기능별로 더 명확히 분리될 필요가 있었다.

## 디자인 구현 프롬프트 위치

```text
docs/safety-features/{feature}/prompts/{NN}_DESIGN_IMPLEMENTATION_PROMPT.md
```
