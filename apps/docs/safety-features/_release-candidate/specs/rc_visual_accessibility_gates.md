# RC Visual & Accessibility Gates

## Webhard

- Drive-like layout 유지
- 좌측 Drive sidebar + 중앙 file canvas
- 상세 패널 기본 닫힘
- ERP 카드형 회귀 금지

## Mailbox

- three-pane mailbox 유지
- account onboarding / connected empty / thread list 상태 분리
- 연결 성공과 계정 없음 동시 표시 금지
- floating compose panel focus/keyboard 확인

- Escape close
- keyboard navigation
- error text association
