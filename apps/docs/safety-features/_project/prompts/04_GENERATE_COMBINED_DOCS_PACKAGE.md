# 04_GENERATE_COMBINED_DOCS_PACKAGE

```text
너는 문서 릴리즈 패키지를 만드는 테크니컬 라이터다.

목표:
Step 01~현재 단계까지의 docs/safety-features 문서를 하나의 combined overlay로 묶을 준비를 하라.

요구사항:
1. docs/safety-features 전체 구조를 검사하라.
2. README, INDEX, registry가 최신인지 확인하라.
3. 기능별 specs/prompts가 있는지 확인하라.
4. 중복/충돌 파일을 확인하라.
5. combined package manifest를 작성하라.
6. 기존 apps/docs/technical-guidance-auto-report는 삭제하지 말고 참조만 유지하라.

완료 기준:
- combined package에 포함될 파일 목록이 있다.
- 제외 파일 목록이 있다.
- 적용 순서와 rollback 기준이 있다.
```
