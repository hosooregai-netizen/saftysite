# 02_VERIFY_COMBINED_DOCS

```text
너는 통합 문서 패키지 검수자다.

목표:
통합 docs package가 기능별 specs/prompts, registry, design-system, quality, release docs를 모두 포함하는지 검증하라.

확인:
- docs/safety-features/INDEX.md
- docs/safety-features/_registry/*
- docs/safety-features/_design-system/specs/*
- docs/safety-features/_quality/specs/*
- docs/safety-features/_release/specs/*
- 각 기능 specs/prompts

완료 기준:
- 누락된 기능 폴더가 없다.
- specs/prompts 분리 원칙이 유지된다.
- registry가 모든 기능을 포함한다.
```
