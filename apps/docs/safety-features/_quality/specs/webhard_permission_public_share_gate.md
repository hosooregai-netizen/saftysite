# Webhard Permission/Public Share Gate

## Release blocking

- public share root 밖 접근 허용
- expired/revoked share가 데이터 반환
- viewer가 edit/delete/share UI 접근 가능
- workspace 밖 item permission 생성 가능
- 권한 없는 사용자에게 dataUrl/textContent/externalUrl 반환
- 웹하드가 ERP 카드형 화면으로 회귀

## Non-blocking

- commenter role이 viewer와 동일 처리
- public editor UI가 아직 숨김
- object storage 전환 미완료
