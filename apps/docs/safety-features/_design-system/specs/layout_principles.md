# Layout Principles

## Layout categories

| Category | Use |
|---|---|
| ERP AppShell | 일반 업무/관리 화면 |
| Fullscreen Workspace Shell | 웹하드, 메일함 |
| Report Workspace | 보고서 작성/검토 |
| List Management | 보고서 목록, 사업장/현장 |
| Settings Flow | 계정/설정 |
| Checkout Flow | 결제 |

## ERP AppShell

```text
Global Sidebar
Main Content
Page Header
Panel/List/Form
```

## Fullscreen Workspace Shell

```text
Topbar
Feature Sidebar
Main Canvas
Optional Detail Panel
```

## Responsive rules

- Desktop: full layout
- Tablet: sidebar collapsible
- Mobile: drawer/stack
- Table/List columns collapse into primary row + metadata
