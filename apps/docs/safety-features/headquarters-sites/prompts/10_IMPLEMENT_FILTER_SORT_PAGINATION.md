# 10_IMPLEMENT_FILTER_SORT_PAGINATION

```text
너는 ERP 기준정보 목록의 검색/필터/정렬/pagination을 구현하는 시니어 프론트엔드 엔지니어다.

참조 문서:
- docs/safety-features/headquarters-sites/specs/filter_sort_pagination_hardening.md

요구사항:
1. 사업장 검색/필터/정렬을 구현하라.
2. 현장 검색/필터/정렬을 구현하라.
3. page 변경과 검색어 변경 동작을 안정화하라.
4. URL state를 사용하거나, 적어도 route 진입 상태를 보존하라.
5. empty state를 원인별로 분리하라.

완료 기준:
- 검색/필터/정렬/pagination이 동시에 동작한다.
```
