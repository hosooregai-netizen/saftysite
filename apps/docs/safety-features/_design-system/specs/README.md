- 기능마다 카드/여백/버튼 스타일이 흔들리는 문제
- 웹하드와 메일함처럼 독립 앱 같은 화면이 ERP AppShell과 충돌하는 문제

- AI 구현 시 “예쁘게”라는 모호한 지시로 UI가 달라지는 문제
