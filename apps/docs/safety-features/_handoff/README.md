# Final Documentation Handoff

`_handoff/`는 프로젝트 문서를 실제 역할별로 빠르게 사용할 수 있게 하는 최종 안내서다.

## 사용 대상

- 신규 개발자
- 프론트엔드 개발자
- 백엔드 개발자
- QA 담당자
- 운영 담당자
- 제품/기획 담당자
- AI 코딩 에이전트
