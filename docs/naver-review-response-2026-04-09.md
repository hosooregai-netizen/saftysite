# 네이버 로그인 재검수 제출 메모

## 서비스 요약

- 서비스명: 대한안전산업연구원 RegAI
- 서비스 유형: 건설기술재해예방지도 운영 ERP형 서비스
- 주요 기능:
  - 관리자/관제 대시보드
  - 사업장·현장 관리
  - 엑셀 업로드
  - 기술지도·분기 보고서 작성 및 출력
  - 메일함 연동 및 발송 관리
  - 지도요원 모바일·현장 업무 지원

## 이번 보완 내용

1. 서비스 URL에서 서비스 콘텐츠를 바로 확인할 수 있도록 공개 소개 페이지를 추가했습니다.
   - 서비스 URL: `https://saftysite-seven.vercel.app/service-intro`
2. 서비스 소개 자료는 `대한안전산업연구원 RegAI` 기준으로 서비스 유형, 메뉴별 기능, 메일 연동 목적이 보이도록 보완했습니다.
3. 네이버 로그인 검수용 단계별 캡처 자료를 별도로 준비했습니다.

## 검수 신청 시 소명 내용 예시

대한안전산업연구원 RegAI는 건설기술재해예방지도 운영을 위한 ERP형 서비스입니다.  
사업장·현장 관리, 기술지도 보고서 작성, 문서 출력, 메일 발송 및 수신 흐름을 하나의 서비스 안에서 제공합니다.

이번 재검수에서는 아래 자료를 함께 제출드립니다.

- 서비스 소개 자료
- 공개 서비스 소개 URL (`/service-intro`)
- 네이버 로그인 버튼 노출 화면
- 네이버 로그인 진입 화면
- 로그인 완료 후 메일함/연결 완료 확인 화면

네이버 로그인은 메일함 기능에서 네이버 메일 계정을 연결하고, 서비스 내에서 수신 메일 확인 및 보고서 메일 발송 흐름을 이어가기 위한 인증 수단으로 사용됩니다.

## 제출 첨부물 체크리스트

- 서비스 소개 PDF
- `01` 네이버 로그인 버튼 노출 화면
- `02` 네이버 로그인/동의 화면
- `03` 추가 정보 입력 화면이 있다면 해당 화면
- `04` 로그인 완료 후 서비스 진입 화면
- `05` 메일함 또는 연결 완료 확인 화면

## 캡처 생성 스크립트

```bash
npm run capture:naver-review
```

환경변수:

- `NAVER_REVIEW_PUBLIC_BASE_URL`
- `NAVER_REVIEW_APP_BASE_URL`
- `NAVER_REVIEW_SERVICE_EMAIL`
- `NAVER_REVIEW_SERVICE_PASSWORD`

출력 경로:

- `.tmp-ui/generated/naver-review/`
