# Batch 06. Landing Conversion Funnel

## 요구사항

- 신뢰형 B2B 히어로
- 4단계 프로세스 소개
- 무료 2건 CTA

## 계약

- `apps/web/app/page.tsx`
- `apps/web/app/pricing/page.tsx`
- `apps/web/app/reports/new/page.tsx`

## 입출력 예시

- 입력: 방문자 랜딩 유입
- 출력: `무료로 2건 시작하기` CTA 클릭, 문서 생성 진입

## 검증

- 모바일/데스크톱 레이아웃 확인
- CTA와 검토 에디터 샘플 진입 확인

## 잔여 리스크

- 실제 인증/결제 연결 전까지는 데모 이동 중심의 전환 퍼널이다.
