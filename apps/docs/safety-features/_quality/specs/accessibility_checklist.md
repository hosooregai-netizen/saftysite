# Accessibility Checklist

## 공통

- [ ] icon-only button에 `aria-label`
- [ ] modal/dialog focus trap
- [ ] Escape로 dialog 닫기
- [ ] keyboard tab order
- [ ] form error와 input 연결
- [ ] loading/error state 텍스트 제공

## 기능별

- Webhard: Enter로 폴더 열기, context menu keyboard 접근
- Mailbox: thread row keyboard selection, compose panel focus
- Report: review queue 항목에서 섹션 이동, disclaimer checkbox label
