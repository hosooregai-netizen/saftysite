# Foundations

## Color

| Token | Purpose |
|---|---|
| `--color-bg` | 전체 배경 |
| `--color-surface` | 카드/패널 배경 |
| `--color-surface-muted` | 보조 영역 |
| `--color-border` | 기본 border |
| `--color-border-strong` | 강조 border |
| `--color-text` | 기본 텍스트 |
| `--color-text-muted` | 보조 텍스트 |
| `--color-primary` | 주요 액션 |
| `--color-primary-soft` | 선택/활성 배경 |
| `--color-success` | 성공 |
| `--color-warning` | 경고 |
| `--color-danger` | 위험/삭제 |
| `--color-info` | 정보 |

## Typography

| Token | Usage |
|---|---|
| `--font-size-xs` | badge, meta |
| `--font-size-sm` | table, helper |
| `--font-size-md` | body |
| `--font-size-lg` | panel title |
| `--font-size-xl` | page title |
| `--font-size-display` | hero title |

## Spacing

4px base scale을 사용한다.

```text
4 / 8 / 12 / 16 / 20 / 24 / 32 / 40 / 48 / 64
```

## Radius / Border / Shadow

- 업무 화면은 thin border 우선
- shadow는 floating layer, dropdown, compose panel에만 제한
- nested shadow 금지
